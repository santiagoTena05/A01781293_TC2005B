HMTL Cheat Sheet

1. a - Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

2. p

Es un elemento semantico que sirve para identificar párrafos.

3. h1, h2, h3, h4, h5, h6

Los headings son son elementos que se usan para dar jerarquías dentro del documento. No deben ser utilizados para fines estéticos (hacer más grande el texto). La mayor prioridad es de h1 al h6.

4. div

El div se usa para identificar grandes agrupaciones de contenido.

5. span

El span se usa para dar estructura inline o dentro de linea esto es la parte del span dentro del párrafo.

6. Strong

Para utilizar negritas y al mismo tiempo dar má importancia.

7. i

Se usa para escribir en itálicas.

8. u

Se usa para mostrar el texto subrayado.

9. b

Similar al strong se usa para escribir en negritas pero con menor énfasis.

10. Strike

Se usa para mostrar el texto tachado con una linea horizontal .

11. Small

El elemento small se usa para reducir el tamaño del texto.

12. em

Similar a lo que sucede con strong y b. Este elemento hace el texto en itálicas pero con un mayor énfasis. a comparación del i

13. Header

El elemento header ayuda para dar estructura a la página web. Se usa para identificar el principio de una página. Puede incluir elementos de navegación. (Es un párrafo dentro de un header)

14. Section

Siempre deben llevar un header

Representa una sección dentro de un documento.

15. Article

El elemento de article se usa para identificar una sección con contenido independiente independiente. De igual manera que el elemento section, article se usa para dar estructura al documento.

16. Aside

Aside se usa para representar una sección que consiste en información que está indirectamente relacionado con el contenido de mayor importancia en el documento.

17. Footer

De igual manera que los 4 elementos anteriores, el elemento footer se utiliza para dar estructura al documento. En específico se utiliza para identificar el final de la página.

18. nav

El elemento nav se usa para distinguir la sección del documento en donde se encuentran los links de navegación de la página.
